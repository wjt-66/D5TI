# 第五人格MBTI测试 - 你的MBTI与哪位第五人格求生者相似？

## 项目简介

这是一个基于MBTI性格测试的趣味网页应用，通过回答与第五人格游戏相关的问题，来测试你的MBTI性格类型，并匹配到最相似的第五人格求生者角色。

## 功能特点

- MBTI性格测试：通过15道与第五人格游戏场景相关的问题，准确测试你的MBTI性格类型
- 角色匹配：根据测试结果，匹配到最相似的第五人格求生者角色
- 详细描述：每个角色都有详细的性格分析和描述
- 角色图片：展示对应求生者的精美图片
- 响应式设计：适配不同设备的屏幕尺寸

## 技术栈

- 前端：HTML5, CSS3, JavaScript (ES6+)
- 样式：自定义CSS，支持响应式布局
- 数据：本地JSON数据存储

## 使用方法

### 在线访问
直接打开 `index.html` 文件即可在浏览器中运行。

### 本地运行
1. 下载项目文件到本地
2. 使用浏览器打开 `index.html` 文件
3. 开始测试你的MBTI性格类型

## 测试流程

1. **回答问题**：回答15道与第五人格游戏相关的问题
2. **获取结果**：系统根据你的选择计算MBTI类型
3. **查看匹配**：查看与你性格最相似的第五人格求生者
4. **阅读分析**：了解详细的性格分析和角色描述

## 开发说明

### 自定义修改

- **添加新问题**：在 `script.js` 中的 `data.questions` 数组添加新的问题
- **修改角色描述**：更新 `poetDescriptions` 对象中的描述内容
- **更换图片**：在 `image/` 目录下替换对应的角色图片
- **样式定制**：修改 `style.css` 文件来自定义界面样式

### 数据格式

每个MBTI类型的数据格式如下：

```javascript
{
    "ISTJ": {
        poet: "守墓人",
        description: "详细描述...",
        imageUrl: "image/守墓人.jpeg"
    }
}
```
